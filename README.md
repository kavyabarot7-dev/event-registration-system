# 🎓 College Event Registration System

A full-stack web application that allows students to register for college events like Tech Fest, Workshops, and Hackathons.

---

## 📌 Project Overview

This project is developed as part of a capstone assignment.  
It provides a simple and efficient way for students to register for events and for administrators to view all registrations.

---

## ✨ Features

- 📝 Student registration form  
- 📊 Admin dashboard to view all data  
- 💾 Data stored in database  
- 🔄 Real-time data update  
- 🎨 Modern UI with animations  

---

## 🛠️ Tech Stack

### Frontend:
- HTML  
- CSS (Glass UI + Animations)

### Backend:
- Node.js  
- Express.js  

### Database:
- MySQL  

---

## ⚙️ How It Works

1. User fills the registration form  
2. Data is sent to backend using POST API  
3. Backend stores data in MySQL database  
4. Admin dashboard fetches data using GET API  
5. Data is displayed in table format
